# Use Cases для Events Web-application

## Use Cases для модели Event

1. **GetAllEventsUseCase** - Получение списка всех событий
2. **GetEventByIdUseCase** - Получение события по ID
3. **GetEventsByTitleUseCase** - Получение событий по названию
4. **CreateEventUseCase** - Создание нового события
5. **UpdateEventUseCase** - Обновление существующего события
6. **DeleteEventUseCase** - Удаление события
7. **GetEventsByCriteriaUseCase** - Получение событий по критериям (дата, место, категория)
8. **AddEventImageUseCase** - Добавление изображения к событию
9. **RemoveEventImageUseCase** - Удаление изображения к событию

## Use Cases для модели User/Participant

1. **RegisterUserForEventUseCase** - Регистрация пользователя
2. **RegisterUserForEventUseCase** - Обновление данных пользователя
3. **RegisterUserForEventUseCase** - Удаление пользователя
4. **RegisterUserForEventUseCase** - вход пользователя
5. **GetEventParticipantsUseCase** - Получение списка участников события
6. **GetParticipantByIdUseCase** - Получение участника по ID
4. **RegisterUserForEventUseCase** - Регистрация пользователя на событие
7. **CancelUserParticipationUseCase** - Отмена участия пользователя в событии
